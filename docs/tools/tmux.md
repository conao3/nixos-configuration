# tmux

prefix: `C-q`

## Quickstart

```sh
tmux
```

ウィンドウを分割して作業する:
- `C-q %` で左右分割
- `C-q "` で上下分割
- `C-q 矢印キー` でペイン移動
- `C-q z` でペインのズーム切り替え

セッションを維持したまま離脱・復帰する:
- `C-q d` でデタッチ
- `tmux a` で復帰

## セッション

セッションは tmux の最上位の単位。複数のウィンドウを持つ。

| コマンド | 説明 |
|----------|------|
| `tmux` | 新規セッション開始 |
| `tmux new -s NAME` | 名前付きセッション開始 |
| `tmux ls` | セッション一覧 |
| `tmux a` | 最後のセッションにアタッチ |
| `tmux a -t NAME` | 指定セッションにアタッチ |
| `tmux kill-session -t NAME` | セッション削除 |

| キー | 説明 |
|------|------|
| `C-q d` | デタッチ |
| `C-q s` | セッション一覧・切り替え |
| `C-q $` | セッション名変更 |

## ウィンドウ

ウィンドウはセッション内のタブ。画面下のステータスバーに一覧表示される。

| キー | 説明 |
|------|------|
| `C-q c` | 新規ウィンドウ |
| `C-q n` | 次のウィンドウ |
| `C-q p` | 前のウィンドウ |
| `C-q NUMBER` | 番号でウィンドウ切り替え |
| `C-q w` | ウィンドウ一覧・切り替え |
| `C-q ,` | ウィンドウ名変更 |
| `C-q &` | ウィンドウ削除 |

## ペイン

ペインはウィンドウ内の分割領域。1つのウィンドウを複数のシェルに分割できる。

| キー | 説明 |
|------|------|
| `C-q %` | 左右分割 |
| `C-q "` | 上下分割 |
| `C-q o` | 次のペインに移動 |
| `C-q 矢印キー` | ペイン移動 |
| `C-q z` | ペインのズーム切り替え |
| `C-q x` | ペイン削除 |
| `C-q C-o` | ペインの位置をローテート |
| `C-q {` | ペインを前に移動 |
| `C-q }` | ペインを後ろに移動 |
| `C-q Space` | レイアウト切り替え |

## コピーモード

スクロールバックやテキスト選択ができるモード。

| キー | 説明 |
|------|------|
| `C-q [` | コピーモード開始 |
| `q` | コピーモード終了 |
| `Space` | 選択開始 |
| `Enter` | 選択コピー |
| `C-q ]` | ペースト |

コピーモード中は vi または emacs キーバインドでカーソル移動できる。

## コマンドモード

| キー | 説明 |
|------|------|
| `C-q :` | コマンドモード |
| `C-q ?` | キーバインド一覧 |
| `C-q t` | 時計表示 |

よく使うコマンド:

| コマンド | 説明 |
|----------|------|
| `resize-pane -D 5` | ペインを下に5行リサイズ |
| `resize-pane -R 10` | ペインを右に10列リサイズ |
| `swap-window -t NUMBER` | ウィンドウの位置を入れ替え |
| `move-window -t NUMBER` | ウィンドウを指定番号に移動 |
